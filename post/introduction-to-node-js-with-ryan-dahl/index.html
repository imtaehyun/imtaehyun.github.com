Introduction to Node.js with Ryan Dahl

Node.js를 만든사람에게서 Node.js에 대해서 들어보자

위 영상은 Node.js를 만든 Ryan Dahl이 MarakanaTechTV에서 Node.js에 대해서 설명하고 몇가지 예제를 보여주는 영상이다.

Node가 0.4버전대 일때의 영상이라 벌써 1년 8개월 전이지만 Node에 대한 본질은 변하지 않았기에 살펴 보았다.

참고로 나는 영어를 원어민처럼 하지 못해서 들리는 부분만 정리해 보았다.

Node.js 란?

Node는 V8이라는 Google이 작성한 매우 복잡한 Virtual Machine 위에 뿌려진 설탕같은 존재이다.

V8은 Chrome 브라우저에 포함된 Javascript Engine이며 이것은 야수(Beast)다. 그것은 매우 똑똑한 사람들에 의해 만들어진 매우 거대하고도 고성능의 VM이다.

Node는 V8의 최상단에 위치한 Library의 집합이며, 효율적인 네트워크를 위해 사용된다고 강조하고 싶다. Node는 Front단의 Javscript 개발자에게는 친숙할 수 있다.

몇가지 예제

Http Server

Http 프로토콜을 이용한 서버

var http = require('http');

var s = http.createServer(function(req, res) {
    res.writeHead(200, {'Content-type' : 'text/plain'});
    res.write("Hello\n");

    setTimeout(function() {
        res.end("world\n");
    }, 2000);
});

s.listen(8000);

Tcp Server

Socket을 이용한 서버

var net = require("net");

var server = net.createServer(function(socket) {
    socket.write('hello\n');
    socket.write('world\n');

    // 연결중 데이터를 수신하면 수신한 데이터를 다시 전송한다.
    socket.on('data', function(data) {
        socket.write(data);
    });
});

server.listen(8000);

위 코드를 실행하면 hello world가 찍히고, 그후에 우리가 치는 내용들이 그대로 다시 출력된다. Echo Server라고 한다.

Chat Server

var net = require('net');

var sockets = [];

var server = net.Server(function(socket) {
    sockets.push(socket);

    // client가 텍스트를 입력하면 자신을 제외한 다른 사람들에게 전달.
    socket.on('data', function(d) {
        for (var i = 0; i < sockets.length; i++) {
            if (sockets[i] == socket) continue;
            sockets[i].write(d);        
        }
    });

    // client가 접속을 종료하면 dead socket을 제거.
    socket.on('end', function() {
        var i = sockets.indexOf(socket);
        sockets.splice(i, 1);
    });
});

server.listen(8000);

Q&A

그럼 뭐가 Node이고, 뭐가 Javascript인가?

Node는 Networking, Http Server, Tcp Server, Printing Standard Out (stdout)의 역할을 하며, 이것들은 Javascript에 의한것이 아니다.

Javascript는 Function, Addition, Numbers, Strings, Array, Object 같은 것들이다.

이러한 요소가 모여서 Node.js를 이룬다.

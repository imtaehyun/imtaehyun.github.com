하루프레스 개인도메인 설정방법

하루프레스를 Github에 올려놓고 쓰고 있는데 Github 페이지를 개인도메인과 연결시킬 수 있다고 해서 시도하다가 몇번 삽질도 하고 해서 기록 및 다른 사람을 위해 기록해본다.

하루프레스 개인도메인 설정방법 (Github Only)

1. 하루프레스 config.js 파일 수정

하루프레스 root 폴더의 config.js파일을 열어 보면 다음과 같이 CNAME을 설정하는곳이 있는데 여기에 자신의 도메인 주소를 입력한다.

"CNAME": "nezz.pe.kr"

2. 하루프레스 deploy

그 후 하루프레스를 make deploy하면 Github Repository에 CNAME파일이 생성된다. 생성된 CNAME파일을 열어보면 1번에서 설정했던 도메인 주소가 나와야 정상이다.
